# Distance Check

This software helps the user see the walking distance from their current location to several locations around Rexburg, as well as giving a path to take and a estimate of the walking time.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Clone repository
2. Install node.js to the local device
3. Use the command "npm install" in the root directory to get the dependencies
4. Create .env with a Google Maps API key, with Maps JavaScript API and Routes API enabled

Instructions for using the software:

1. Start the web server
2. Open the webpage
3. Give location permissions

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Node.js 24.15
* vite 8.0.13*
* react-google-maps 1.8.3*

*Included in package.json
